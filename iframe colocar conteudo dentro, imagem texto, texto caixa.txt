O atributo id dá um identificador único ao <iframe>, que pode ser usado para:
Manipular o iframe com JavaScript;

Aplicar CSS específico para ele.

nome="frame"

Serve para dar um nome interno ao iframe — muito usado quando
links usam o atributo target.

srcdoc="..."

Este é o atributo mais importante aqui.

srcdoc significa “source
document” (documento de origem).
Ele permite que você escreva o HTML diretamente
dentro do atributo, em vez de carregar um arquivo externo via src.

<h1>Escolha uma das oções abaixo</h1>

Essa é uma tag de título principal.

<p> ... </p>

O <p> define um parágrafo de texto.

<img src='cursoemvideo.png'>

Mostra uma imagem dentro do iframe.
Se o arquivo estiver em outra pasta, será preciso ajustar o caminho, por exemplo:
<img src='imagens/cursoemvideo.png'>
